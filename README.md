This repository contains a simple case study performed to meet the requirements of the Bellabeat case study as defined in the Google Data Analytics capstone course as part of the Google Data Analytics course series on Coursera. Please see the attached Jupyter Notebook for further information on the case study and methods used.

Coding Languages Used:
* Python
* SQL (via SQLite)

Data used for the project can be found in the folder *archive* and its relative subfolders. The directory contains FitBit data from a small subset of users over the course of two months. This data was downloaded via Kaggle at the following URL:

https://www.kaggle.com/datasets/arashnic/fitbit
